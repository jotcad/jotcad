For this you will need 6 buttons, some jumpers, two breadboards (if using the classic ESP32, to have more space and make it look nicer), an i2s DAC, an i2c SSD1306 OLED display (also looks nice in the two-color version), and a rotary encoder (I tested with the KY-040), and finally the ESP32 (any dual-core will do). i2s DAC -> ESP { LCK(15), DIN(2), BCK(4)} display -> ESP { SCK(22), SDA(21)} encoder -> ESP { CLK(25), DT(33), SW(32)} buttons -> ESP (connect in your preferred style) {5, 27, 14, 13, 18, 19}
