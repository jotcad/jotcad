# JotCAD Documentation Index

This directory contains specifications, architectural plans, development guidelines, and DSL references for the JotCAD geometric modeler.

## Directory Responsibility
The `/docs` directory serves as the single source of truth for the language grammar, VFS networking, CGAL boolean kernel adapters, and UI layers of JotCAD.
