# JotCAD UX Components

Interactive UI elements for the blackboard.

## Subdirectories

-   `canvas/`: The primary spatial blackboard layer, nodes, and connections.
-   `discovery/`: Mesh network visualization and schema management.
-   `editor/`: Code editors (`JotNode`) and associated result lists.
-   `system/`: Global HUD elements like the console and error overlays.
-   `viewport/`: High-performance 3D viewport containers.
